The project implemented is Home Automation System in the ECGR course 5090 by the group members: Laukik Karnavat, Gautham Bekal, Ajinkya Sangvikar.

Software Requirements for this project include:
Android Studio, Android Phone, Lampp Server, PhpStorm (for writing php scripts in our case), Postman, Remmina Remote Desktop Client, PuTTY, Xming, Netbeans on Raspberry Pi, IP webcam App, VncServer, Virtual Box.


An Android App named "49erSense" allows the login for 3 different type of users depending on their roles.

Customer is a vital part of the implementation as their exist IOT in his home and has actuators/ sensors for various electrical appliances for which we show status in TextView and update it in MySQL database.

For Customer Live Feed, Weather forecast for 5 days is also implemented.

Utility company can access the status of Customer and show data.

Power Generator fetches the data from the MySQL database of Customer and plots a realtime Barchart for the power consumption according to the status of Appliances.

We succesfully implemented 3 different servers.
